Fitness Web Components

This is a demo of the fitness card and activity ring components, with default and alternative styles. You can use a fully featured “fitness card” or just the activity rings. The default activity ring design is inspired by Apple Fitness.

You can download code from the fitness visualisations repository on GitHub.

Web Components

These visualisations are built using Web Components and must be imported before they are rendered. 

import './FitnessCard.js';
import './FitnessRing.js';

Styling

You can control the font and colours of the card and rings using CSS variables. The --fitness-value controls the colour of the values in the footer. 

--fitness-font: sans-serif;

--fitness-value: #9ca3af;
--fitness-move: #fa114f;
--fitness-exercise: #92e82a;
--fitness-stand: #1eeaef;

--fitness-ring-move: #fa114f;
--fitness-ring-exercise: #92e82a;
--fitness-ring-stand: #1eeaef;

You can target the styling of sections within the card component using the :part pseudo-element. 

fitness-card::part(header) {}
fitness-card::part(main) {}
fitness-card::part(ring) {}
fitness-card::part(footer) {}

Cards

To use the card, you must create a <fitness-card> element with the relevant attributes representing the data.

The date is optional, if there is not set, then the card header is not rendered. steps, distance and flights are also optional – if none are provided, then the card footer is not rendered.

The distance attribute should be provided in in meters. By default, this is converted to miles but you can change this to kilometres using units="km" attribute.

<fitness-card
    date="2023-06-04"
    steps="5258"
    distance="2455"
    units="miles"
    flights="3"
    move-total="1800"
    move-goal="1400"
    exercise-total="20"
    exercise-goal="30"
    stand-total="9"
    stand-goal="12"
></fitness-card>

Rings

To use the ring, you must create a <fitness-ring> element with the relevant attributes representing the data.

<fitness-ring
    move-total="1800"
    move-goal="1400"
    exercise-total="20"
    exercise-goal="30"
    stand-total="9"
    stand-goal="12"
></fitness-ring>
